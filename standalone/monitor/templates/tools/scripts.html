로컬 실행 스크립트

유저 PC에서 직접 실행하는 포렌식 수집·분석 스크립트입니다. 브라우저는 로컬 디스크/메모리에 접근할 수 없어 다운로드 후 관리자 권한으로 직접 실행해야 합니다.

실행 전 확인 사항: • 관리자/root 권한 필요 • 본인 소유 시스템 또는 위임받은 시스템에서만 실행 • 결과 파일은 평문이므로 안전하게 보관
다운로드 무결성 검증 체크섬 검증 페이지 checksums.txt
각 스크립트의 SHA-256을 우측에 표시했습니다. 다운로드 후 다음 명령으로 검증하세요:
Windows: Get-FileHash <file> -Algorithm SHA256
Linux: sha256sum <file>
macOS: shasum -a 256 <file>
